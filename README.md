# text-rpg-roguelike

# 🦝 EHOT VS CROISSANT: TEXT ROGUELIKE v1.4

Ультимативный пошаговый симулятор выживания Сверххищника на Помойке.
Спроектировано на чипе Apple M4, спустя 9 дней изученния python, специально для подрыва пердаков казуалов с инфоцыганских курсов.

## 🚀 ИНСТРУКЦИЯ ПО ДЕПЛОЮ И ПУСКУ:

1. Распакуй архив в систему.
2. Кликни дважды по файлу `start.command` (macOS самостоятельно проверит ОЗУ и Python) или `start_windows.bat`, если у тебя Windows.
3. Приятного гринда!

## ⚠️ ИЗВЕСТНЫЕ БАГИ (ФИЧИ):

- Класс "Дурачонок" имеет скрытый баланс отрицательной атаки и лечит Мусорщика укусами.
- Еноты могут убить Мусорщика уже после своей смерти - они не чистят зубы и едят прямо из помойки.
